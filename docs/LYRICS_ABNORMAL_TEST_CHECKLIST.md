# LYRICS_ABNORMAL_TEST_CHECKLIST

Last Updated: 2026-04-14 12:44:35
Status: Draft v1 (T-008)

## Purpose
基于 `docs/LYRICS_STATE_MACHINE.md` 输出歌词异常场景测试清单，作为实现与联调的验证基线。

## Scope
- 嵌入歌词读取异常
- 远程歌词请求异常
- 缓存异常与 TTL 过期
- 切歌取消与防串歌
- UI 降级行为一致性

## Checklist

### A. Embedded Lyrics Path
- [ ] A1 嵌入歌词标签损坏
  - 条件: 音频含损坏的同步歌词标签
  - 预期: 进入远程拉取分支，不阻塞播放
  - UI: 先显示“加载中”，最终显示远程结果或“暂无歌词”

- [ ] A2 嵌入歌词为空字符串
  - 条件: 标签存在但内容为空
  - 预期: 视为 `EmbeddedNotFound`，进入异步远程拉取
  - UI: 不出现错误弹窗

- [ ] A3 嵌入歌词时间轴格式异常
  - 条件: 含无效时间戳
  - 预期: 解析失败后降级远程拉取
  - 日志: 记录解析失败（脱敏）

### B. Remote Fetch Path
- [ ] B1 LrcApi 超时
  - 条件: 请求超时（`lyrics.fetch_timeout_ms`）
  - 预期: 转 `NoLyrics`，播放不中断
  - UI: 显示“暂无歌词”

- [ ] B2 LrcApi 返回 5xx
  - 条件: 服务端异常
  - 预期: 记日志并降级
  - UI: 不阻塞、不反复弹提示

- [ ] B3 LrcApi 返回空结果
  - 条件: 请求成功但无匹配歌词
  - 预期: 进入 `NoLyrics`
  - UI: 明确空态提示

- [ ] B4 网络断开
  - 条件: 启动后断网
  - 预期: 远程拉取失败降级，不影响当前播放
  - 能力状态: 标记“歌词远程补全不可用”

### C. Cache Path
- [ ] C1 内存缓存命中
  - 条件: 同曲目重复播放
  - 预期: 直接命中展示，不发起远程请求

- [ ] C2 磁盘缓存命中且 TTL 有效
  - 条件: 本地有有效缓存
  - 预期: 展示缓存并回填内存缓存

- [ ] C3 磁盘缓存过期
  - 条件: TTL 到期
  - 预期: 按 miss 处理并触发远程拉取
  - 待确认: 远程失败是否回退过期缓存（B-003）

- [ ] C4 缓存读取失败
  - 条件: 文件损坏或权限问题
  - 预期: 记日志并继续后续分支，不中断播放

### D. Concurrency and Track Switch
- [ ] D1 请求中切歌
  - 条件: 远程请求未返回时切换下一曲
  - 预期: 旧请求被取消，结果不更新到新曲目
  - 校验: 不串歌

- [ ] D2 高频切歌
  - 条件: 连续快速触发 next/prev
  - 预期: 请求取消链稳定，无崩溃/无卡死

### E. UI and Capability Fallback
- [ ] E1 加载态可见
  - 条件: 切到无缓存新曲目
  - 预期: 先展示“歌词加载中”

- [ ] E2 降级态一致
  - 条件: 远程失败
  - 预期: 统一显示“暂无歌词”

- [ ] E3 多语言优先级
  - 条件: 返回多语言内容
  - 预期: 中文优先，其次原文

### F. Logging and Safety
- [ ] F1 错误日志脱敏
  - 条件: 任意异常
  - 预期: 不输出 token/敏感字段明文

- [ ] F2 日志分级
  - 条件: 解析失败/超时/空结果
  - 预期: 使用可区分级别（warn/error），便于排查

## Test Data Suggestions
- 1 首含正常嵌入歌词曲目
- 1 首含损坏/空嵌入歌词曲目
- 1 首无嵌入歌词曲目
- 1 组多语言歌词返回样本

## Exit Criteria
- A/B/C/D/E/F 各分组关键场景至少通过 1 次。
- 无“歌词串歌”与“播放被歌词流程阻塞”问题。
- 异常场景均有明确 UI 反馈与日志记录。

## Open Dependencies
- B-003: 远程失败时是否回退过期缓存、是否快速重试（待确认）。
